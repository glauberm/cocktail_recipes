# Cocktail Recipes

An Angular (7.1.1) app that searchs for recipes from the [TheCocktailDB](https://www.thecocktaildb.com/).

## Installing

Clone the repository and then install the dependencies:

```[bash]
npm install && npm start
```

## Running development server

```[bash]
ng serve
```

## Running tests

```[bash]
ng test
```
